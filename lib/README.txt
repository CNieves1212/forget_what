The following are instructions for booting up the app on an Android Phone Emulator through Android Studio: 

Setting up Virtual Device
    1. Download and open Android Studio version 3.0 or newer (https://developer.android.com/studio?gclid=Cj0KCQjw7pKFBhDUARIsAFUoMDaU8t1qq5PFJGlDmkwGOaR3MS4E9mKE7we8gtYMuAjbWAnzark8h_caAnp8EALw_wcB&gclsrc=aw.ds#downloads)
    2. Go to Tools->AVD Manager->Create Virtual Device and create a new device (App was tested on Pixel 3a with an x86 Image Android 11.0)
    3. Boot up virtual device from Android Studio
Setting up project in Android Studio (instructions derrived from https://flutter.dev/docs/get-started/editor)
    1. Go to Preferences->Plugins, select 'Flutter' and follow install process.
    2. Restart Android Studio if needed
    3. Download/Clone repository from GitHub
